# LEC - Alpha

This repository contains the database design, documentation, and backend
structure for the Love Economy Church Alpha Branch hierarchical leadership and membership system.

## System Purpose

The system is designed to manage church leadership hierarchy, member records,
roles, and profile images in a secure and scalable manner.

## Hierarchy Structure

- Zonal Head
- MC Heads
- Buscenta Heads
- Cell Shepherds
- Shepherds
- Members

## Core Features

- Hierarchical organizational structure
- Member and leadership role management
- Profile image storage
- Secure data access using Supabase (PostgreSQL)

## Technology Stack

- Supabase (PostgreSQL)
- GitHub for version control

## Status

This system is currently under active development.
